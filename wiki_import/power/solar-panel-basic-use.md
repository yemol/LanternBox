# 太阳能板基础使用

## 适用场景

适用于停电、外出、离网供电、移动电源补电、Field Terminal 或小型设备太阳能充电的情况。

这篇内容用于提高太阳能板使用效率，避免低效摆放和错误连接。

## 核心原则

1. 太阳能板需要光照和角度。
2. 阴影会明显降低输出。
3. 优先给储能设备充电，再给终端设备供电。
4. 连接前确认电压、电流和接口。
5. 太阳能输入不稳定时，不建议直接供关键设备长期运行。

## 摆放位置

优先选择：

- 阳光直射处
- 避免阴影处
- 通风位置
- 不易被踩踏的位置
- 不易进水的位置
- 方便调整角度的位置

避免：

- 树影
- 栏杆阴影
- 窗内弱光
- 高温密闭区域
- 潮湿地面
- 线材横跨通道

## 操作步骤

### 1. 检查设备

检查：

- 太阳能板是否破损
- 接口是否匹配
- 线材是否完好
- 控制器是否正常
- 移动电源是否支持太阳能输入

### 2. 摆放

让太阳能板尽量朝向强光，减少遮挡。

### 3. 连接

优先连接：

1. 太阳能板
2. 控制器或充电模块
3. 移动电源或电池
4. 终端设备

不要把不匹配的电压直接接入设备。

### 4. 记录输出

记录：

- 天气
- 摆放位置
- 开始时间
- 结束时间
- 充入电量
- 异常发热

## 充电优先级

优先：

1. 通讯设备
2. 照明设备
3. Core 辅助电源
4. 记录设备
5. 舒适性设备

## 检查点

每次使用检查：

- 是否有阴影
- 线材是否松动
- 设备是否发热
- 输出是否稳定
- 移动电源是否正在充电
- 是否有进水风险
- 是否影响通行

## 常见错误

- 放在阴影里还以为在充电。
- 直接给关键设备供电，太阳一遮就断。
- 接口不匹配硬插。
- 线材绊倒人。
- 雨天不收设备。
- 不记录实际充电效果。

## 停止条件

出现以下情况时，应停止使用：

- 接口发热
- 线材破损
- 控制器异常
- 电池鼓包
- 有焦味
- 下雨进水
- 风大可能吹落
- 输出设备报错

## 备注

壳中灯能源面板未来应记录太阳能输入、移动电源状态和关键设备续航。

太阳能不是魔法电泉，它更像慢慢滴水的屋檐。用得好，要靠角度、耐心和优先级。
