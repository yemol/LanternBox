# 伤口清洁与观察基础

## 用途

介绍小型割伤、擦伤和浅表伤口的基础清洁、覆盖和观察方法，重点是减少污染、识别感染迹象，并明确需要停止自行处理的情况。

## 适用场景

适用于小型割伤、擦伤、浅表皮肤破损、轻微工具划伤等情况。

不适用于严重出血、深部穿刺伤、大面积撕裂伤、动物咬伤、污染严重伤口、烧伤、骨折外露或已经出现明显感染的伤口。

这篇内容用于在无法立即获得专业医疗支持时，帮助进行基础清洁、覆盖和观察，降低感染风险。

## 操作步骤

### 核心原则

1. 先止血，再清洁。
2. 用清洁水冲洗伤口，减少污物和污染。
3. 不要把刺激性消毒剂直接倒入伤口深处。
4. 清洁后覆盖伤口，减少再次污染。
5. 每天观察变化。
6. 一旦出现感染或严重伤情迹象，应停止自行处理并升级处置。

### 1. 先保护自己

处理伤口前：

1. 洗手。
2. 如果有一次性手套，先戴手套。
3. 准备清洁水、干净纱布、绷带或创可贴。
4. 如果环境脏乱，先把伤者移到更干净、更安全的位置。

### 2. 止血

如果伤口在流血：

1. 用干净纱布或干净布料直接按压。
2. 保持稳定压力。
3. 不要频繁掀开查看。
4. 如果布料被血浸透，不要马上拿掉，可在外面再加一层继续按压。

少量渗血通常可以通过按压逐渐减轻。

### 3. 清洁伤口

出血控制后：

1. 用清洁流动水冲洗伤口。
2. 清洗伤口周围皮肤。
3. 可用温和肥皂清洗伤口周围，但避免把肥皂大量弄进伤口内部。
4. 如有小颗粒污物，可用清洁镊子轻轻移除。
5. 如果污物无法去除，不要强行深挖。

不要把双氧水、碘酒或高浓度酒精直接倒入伤口内部。它们可能刺激组织，影响愈合。碘伏等消毒剂更适合用于伤口周围皮肤或按产品说明使用。

### 4. 覆盖伤口

清洁后：

1. 用干净纱布、创可贴或无菌敷料覆盖。
2. 固定时不要缠得太紧。
3. 保持覆盖物清洁、干燥。
4. 如果敷料湿了、脏了或脱落，应及时更换。
5. 每次换药时观察伤口变化。

### 5. 记录观察

建议记录：

- 受伤时间
- 受伤原因
- 伤口位置
- 是否接触泥土、污水、铁锈或动物
- 是否清洗
- 是否持续疼痛
- 是否肿胀
- 是否出现分泌物
- 体温变化

在壳中灯系统中，可将伤口观察写入日记记录，类型选择“健康状况”。

## 判断标准

### 检查点

每日检查：

- 疼痛是否减轻
- 红肿是否扩大
- 是否有脓液或异常分泌物
- 伤口周围是否变热
- 是否出现发热
- 敷料是否干净
- 是否有异味
- 活动时是否牵拉伤口

如果伤口逐渐变干净、疼痛减轻、没有扩大红肿，一般说明恢复方向较好。

## 风险提示

### 常见错误

- 没洗手就处理伤口。
- 用脏布随便包扎。
- 反复揭开伤口查看。
- 伤口还脏就直接包起来。
- 把刺激性液体直接倒进伤口。
- 敷料湿了或脏了还继续使用。
- 伤口恶化后仍然只换创可贴。

### 停止条件

出现以下情况时，不应继续只做家庭处理，应升级处理：

- 出血持续不止
- 伤口很深或边缘张开
- 伤口内有无法清除的异物
- 被动物或人咬伤
- 被明显污染物、污水、泥土、铁锈物划伤或刺伤
- 疼痛持续加重
- 红肿范围扩大
- 出现脓液
- 伤口周围发热
- 出现发烧
- 出现红线向外扩散
- 手指、脚趾麻木或活动受限
- 伤者有糖尿病、免疫力低下或其他高风险情况

如果怀疑破伤风风险，尤其是脏污伤口、深刺伤或疫苗状态不明，应尽快寻求专业医疗建议。

## 记录建议

### 备注

这篇内容只用于小型伤口基础处理，不替代专业医疗。

在资源不足时，最重要的是：

1. 保持伤口清洁。
2. 减少再次污染。
3. 每天观察变化。
4. 发现恶化就停止自行处理并升级。

建议物资库存中准备：

- 一次性手套
- 纱布
- 绷带
- 医用胶带
- 创可贴
- 生理盐水
- 碘伏
- 清洁镊子
- 体温计
- 记录本或日记模块
