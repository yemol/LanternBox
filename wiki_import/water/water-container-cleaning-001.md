# 储水容器清洁基础

## 用途

介绍饮用水储存容器在使用前的清洗、消毒、标记和停用条件，降低储水二次污染风险。

## 适用场景

适用于准备家庭饮用水储备、重复使用水桶、水壶、折叠水袋或饮水容器之前。

这篇内容用于避免“水本身安全，但容器把水污染”的情况。

## 准备材料

### 容器选择

优先选择：

- 未开封瓶装水原容器
- 食品级储水桶
- 饮用水桶
- 干净可密封水壶
- 合格折叠水袋

不要使用：

- 装过农药的桶
- 装过燃油的桶
- 装过清洁剂的桶
- 有强烈异味的容器
- 无法密封的容器
- 内壁难以清洁的旧容器

## 操作步骤

### 核心原则

1. 饮用水容器必须先清洗再储水。
2. 容器应为食品级或明确适合饮用水接触。
3. 曾装过化学品、燃油、农药或清洁剂的容器不要用于饮用水。
4. 清洁后的容器应加盖、防尘、标记日期。
5. 有异味、破损、油膜或来源不明的容器应停用。

### 1. 初步检查

检查：

- 是否破损
- 是否漏水
- 是否有异味
- 是否有油污
- 是否有沉积物
- 是否能密封

### 2. 清洗

1. 用清水冲洗容器。
2. 用洗洁剂或肥皂水清洗内壁。
3. 用干净刷子清洁边角和瓶口。
4. 完全冲洗干净，避免残留泡沫。

### 3. 消毒

可使用适合饮用水容器的消毒方法。

使用含氯消毒剂时，应确认产品浓度和使用说明，避免使用带香味、带清洁添加剂或不适合饮用水容器的产品。

消毒后应让消毒液接触容器内表面，再彻底处理并避免残留异味。

### 4. 标记

容器外部标记：

- 编号
- 容量
- 清洗日期
- 储水日期
- 用途：饮用水
- 存放位置

## 判断标准

### 检查点

储水前检查：

- 容器无异味
- 容器无油膜
- 容器无裂缝
- 盖子可密封
- 已贴标签
- 清洗日期清楚
- 不与非饮用水容器混放

## 风险提示

### 常见错误

- 用旧清洁剂桶装饮用水。
- 容器只冲一下就储水。
- 忘记清洗瓶口和盖子。
- 饮用水桶和清洁水桶混用。
- 容器没有标签。
- 把储水桶放在化学品旁边。

### 停止条件

出现以下情况时，容器不应继续作为饮用水容器：

- 有持续异味
- 有油膜
- 有裂缝或漏水
- 曾装过毒性物质
- 内壁发霉
- 容器来源不明
- 无法彻底清洁
- 盖子无法密封

## 替代方案

没有食品级容器时，把可疑容器仅用于非饮用水，并清楚标记；没有刷子时用少量清洁水反复摇洗瓶口、盖子和内壁。曾装过燃油、农药、清洁剂或有持续异味的容器，不用于饮用水。
