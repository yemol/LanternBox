# 成员信息备份

## 是什么

成员信息备份记录身份、联系方式、过敏、慢病、技能和紧急联系人。它属于“信息保存与长期重建”主题，用来帮助小团队在离线、断供或长期恢复阶段形成一致判断。

## 为什么重要

- 在设备损坏、断电、人员更替或压力升高时，保留可读、可交接的依据。
- 让证件、医疗、库存、维修、种植和事件记录能够被后来的人复核。
- 减少只靠记忆带来的遗漏、争执和重复试错。

## 如何判断

- 检查“成员信息备份”是否写清对象、日期、责任人和版本。
- 确认纸质或离线副本能在无网络、低电量或设备更换后读取。
- 把亲眼确认、他人转述、推测和待验证内容分开标注。
- 抽查一名未参与记录的人，能否按记录找到资料或理解下一步。

## 适用边界

- 适用于长期档案、交接、复盘和低频但关键的信息保存。
- 涉及隐私、医疗和身份资料时，只给需要照护或决策的人查看。
- 资料已经潮湿、残缺或来源不明时，应先标注风险，不直接覆盖旧记录。

## 常见误区

- 只保存电子版，没有纸质摘要或恢复说明。
- 不断覆盖旧文件，发现新版本损坏时没有回退点。
- 记录很多但没有索引，紧急时找不到。
- 把未经验证的传言写成确定事实。
