# 手工具编号和交接记录

## 用途

让刀、钳、螺丝刀、锯、手电、扳手和手钻有固定编号、存放点和责任记录，减少“用完找不到”的维修中断。

## 适用场景

适用于多人共用工具、外出携带、借用归还和工具箱重整。不适用于外部交换风险判断和危险工具权限审批。

## 准备材料

最低条件是标签、记号笔、记录纸和工具清单。条件允许时准备耐磨标签、分格盒、照片或手绘工具轮廓。

## 操作步骤

1. 按工具类别编号，例如 LB-TOL-PLI-001 表示第一把钳子。
2. 在工具和存放位都写编号，尖锐工具标签不贴在握持受力处。
3. 建立借用栏：借用人、用途、取出时间、预计归还和归还检查。
4. 每日或每次任务后按编号归位，缺失立即标记。
5. 对禁用工具保留编号，但移到待修区，避免再次领用。

## 判断标准

- 合格：工具编号可读，存放位明确，借用和归还能追溯。
- 需修正：标签磨损、工具一物多名、工具箱里有未编号件。
- 停止：危险工具无人负责、借出后状态不明或待修工具混入可用区。

## 风险提示

停止条件是工具涉及刀具、电气、承重或外借却没有责任人。不要因为记录麻烦就让关键工具无编号流动。

## 替代方案

缺标签时，用绳结颜色、刻痕、纸卡和存放位轮廓替代；缺表格时在日记中固定一页工具清单。

## 记录建议

记录编号规则、初始数量、存放位置、借用人、归还状态和损坏说明。
