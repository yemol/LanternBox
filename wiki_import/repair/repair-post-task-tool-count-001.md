# 维修任务结束后工具清点

## 用途

维修结束后核对所有工具、材料和废料，确认刀片、钉子、钻头、钳子、锯和禁用工具没有遗留在地面、桌下、口袋或现场角落。

## 适用场景

适用于任何多人或危险工具维修任务，尤其是低光、外出、儿童会经过和现场空间小的场景。不适用于人员正在受伤需立即处理的阶段。

## 准备材料

最低条件是任务前清单、笔、废料盒、工具箱和禁用标签。条件允许时准备磁铁、手电、归还检查区和异常袋。

## 操作步骤

1. 按任务前清单逐件核对工具编号和数量。
2. 检查工具状态，发现裂纹、缺口、松动、发热或异响就贴禁用标签。
3. 搜查桌面、桌下、地面、废料盒边缘、口袋和材料堆。
4. 把废刀片、钉子、尖木屑和铁丝头放入固定容器。
5. 负责人确认清点结果后，才解除工具区边界。

## 判断标准

- 合格：工具数量一致，废料收齐，禁用工具已隔离。
- 需调整：只有大工具归还，小刀片、钉子和钻头未核对。
- 停止：工具数量对不上、地面仍有锐器、禁用标签脱落或儿童准备进入现场。

## 风险提示

工具清点不是收拾卫生。刀片、钉子和钻头遗漏会导致后续踩伤、割伤和误用；数量对不上时不得宣布任务结束。

## 替代方案

缺磁铁时用手电低角度照地面找反光物；缺工具箱时用纸盒分区归还；缺负责人时由两人交叉清点。

## 记录建议

记录任务名称、结束时间、归还工具编号、缺失项、损坏项、废料处理和复查人。
