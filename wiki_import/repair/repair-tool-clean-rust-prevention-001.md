# 工具使用后的清洁和防锈

## 用途

把工具从“刚用完”恢复到可交接状态，减少泥水、胶层、锈粉、木屑和金属屑造成的卡滞、钝化和污染。

## 适用场景

适用于钳子、螺丝刀、刀具、锯片、手钻、扳手和小夹具使用后。不适用于已接触污水、血液、化学品或带电损坏的工具，这类先隔离。

## 准备材料

最低条件是干布、小刷子、工具盒、标签和记录纸。条件允许时准备少量防锈油、砂纸、废油布和干燥剂。

## 操作步骤

1. 先清掉木屑、金属屑、泥沙和胶带残胶。
2. 擦干工具，活动关节来回开合，确认无卡滞和异响。
3. 金属面有浮锈时轻刷，不把锈粉带进干净工具盒。
4. 刀具、锯片和尖锐件套上护套或单独朝同一方向放置。
5. 发现松动、崩口、裂柄和污染时贴标签，不回到可用区。

## 判断标准

- 可归位：干燥、无残胶、活动顺畅、编号可读、刃口有保护。
- 待维护：轻锈、残胶、关节发涩或编号脱落。
- 隔离：污水污染、血迹、不明化学味、刃口崩裂或使用中失效。

## 风险提示

停止条件是工具污染来源不明、清洁后仍有异味、关节夹住金属屑或锐器没有护套。不要把湿工具直接放进封闭盒。

## 替代方案

缺防锈油时，先做到干燥、通风、分层存放；用干布包住关键金属面，定期打开检查。

## 记录建议

记录工具编号、使用任务、清洁人、异常、维护需求和归位时间。
