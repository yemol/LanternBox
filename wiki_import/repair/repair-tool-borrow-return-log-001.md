# 工具借用归还记录

## 用途

记录团队内部谁拿走了什么工具、做什么、何时归还、归还时是否损坏，保证关键维修能力可追踪。

## 适用场景

适用于多人共用工具箱、外出维修、夜间值守携带和临时借给相邻小组。不适用于外部高风险交换决策本身。

## 准备材料

最低条件是工具编号、借用表、笔和归还检查点。条件允许时准备工具影子板、借用牌和禁借清单。

## 操作步骤

1. 借出前写工具编号、借用人、用途、取出时间和预计归还时间。
2. 唯一关键工具、危险工具和待修工具默认不借，除非负责人批准。
3. 归还时检查缺件、裂纹、锈蚀、污染、刀片状态和电量。
4. 异常工具放入待修区，不直接回到可用位置。
5. 未按时归还时先查任务状态，再更新工具位置。

## 判断标准

- 合格：每件借出工具能找到人、用途、归还状态和复查结果。
- 需修正：只有口头借用，没有编号和时间。
- 停止：危险工具去向不明、归还后损坏未标记或唯一工具被借出过夜。

## 风险提示

停止条件是刀具、电气工具、照明、绳索和钳子被无记录拿走。不要让待修工具通过归还流程混回工具箱。

## 替代方案

缺表格时，用门口纸卡或工具箱盖内侧写临时借用栏；缺编号时先按位置和外观描述。

## 记录建议

记录工具编号、借用人、用途、取还时间、归还状态、损坏说明和是否允许再次借用。
