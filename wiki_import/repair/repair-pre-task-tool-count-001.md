# 维修任务开始前工具点数

## 用途

在维修任务开始前点清需要的工具、材料和禁用物，避免做到一半才发现缺锯、缺钳子、拿到破损工具或把危险工具遗留在儿童附近。

## 适用场景

适用于修补容器、固定棚布、切割材料、拆解旧物和外出维修。不适用于紧急人员救护；救护完成后再补记录。

## 准备材料

最低条件是工具清单、笔、工具编号和临时放置盒。条件允许时准备任务卡、借用牌、材料清单和禁用工具栏。

## 操作步骤

1. 写清本次任务需要的工具，不把“可能用到”都带走。
2. 按编号点数，确认每件工具在可用区而不是待修或禁用区。
3. 检查刀口、锯齿、钳口、钻头、手电和胶带状态。
4. 把不需要的危险工具留在隔离区，减少现场混乱。
5. 任务开始前由负责人读一遍工具数量和归还地点。

## 判断标准

- 合格：工具数量、编号、状态和归还点清楚。
- 需调整：工具混在材料中，或借用人不知道哪些要归还。
- 停止：关键工具缺失、禁用工具混入、危险工具多带或照明不足以检查状态。

## 风险提示

不要为了快开始而跳过点数。工具数量对不上时，不开工；危险工具多带到现场会增加误拿和丢失风险。

## 替代方案

缺清单时用纸卡写任务必需三到五件工具；缺编号时按外观和位置临时描述；缺负责人时只做低风险准备，不开危险工具任务。

## 记录建议

记录任务名称、负责人、工具编号、初始数量、状态异常、禁用工具和预计归还时间。
