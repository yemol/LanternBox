# 简易夹具和临时固定

## 用途

在切割、打孔、胶粘、绑扎和打磨前，用低资源方式把材料固定住，减少一只手扶材料、一只手用工具造成的滑移和割伤。

## 适用场景

适用于低力加工、小件定位、胶水固化和临时压紧。不适用于人员承重、弹性件释放、带电设备、玻璃破片和高力撬动。

## 准备材料

最低条件是木块、绳子或布条、重物、纸板垫和标记笔。条件允许时准备 C 形夹、橡胶垫、楔子和工作板。

## 操作步骤

1. 判断材料需要防滑、防转、防翘还是保持角度，不同目标分开固定。
2. 在材料和压紧物之间垫布或纸板，避免夹出深压痕。
3. 用两处以上固定点限制移动，绳带绕过锐边时加护垫。
4. 轻推测试，观察是否滑移、翘起、旋转或发出裂响。
5. 加工后先解除工具，再慢慢释放夹持，检查材料是否回弹。

## 判断标准

- 可用：材料在低力推拉下不移动，固定点不割绳，压痕不影响用途。
- 调整：绳带拉长、木楔滑出、重物晃动、材料边缘被压白。
- 停止：必须用手靠近工具路线扶住，或固定点一松就会砸落、弹出。

## 风险提示

停止条件是固定物滑移、绳带磨断、材料裂开、重物可能滚落或加工方向指向身体。不要把人体重量作为夹具。

## 替代方案

缺少绳带时，用木楔、门缝、抽屉边和重物做低力固定；仍不稳定时改为两人协作，但扶持者手必须离开工具路线。

## 记录建议

记录临时夹具做法、使用材料、可承受动作、是否留下压痕和需要改进的位置。
