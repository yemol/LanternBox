# 小工具包基础配置

## 适用场景

适用于家庭维修、设备维护、停电处理、外出采集、Core 和终端设备维护、物资整理和轻度应急修补。

这篇内容用于建立一个基础、可携带、可维护的小工具包。

## 核心原则

1. 工具包应服务于常见小故障。
2. 不追求多，而追求常用、可靠、找得到。
3. 工具和耗材要一起准备。
4. 使用后要归位。
5. 工具包也需要定期检查。

## 基础配置

### 1. 切割类

建议准备：

- 小剪刀
- 美工刀
- 折叠小刀
- 备用刀片

用途：

- 剪胶带
- 拆包装
- 削线皮
- 切割绳子

### 2. 固定类

建议准备：

- 布基胶带
- 电工胶带
- 扎带
- 魔术扎带
- 细绳
- 小夹子

用途：

- 临时固定
- 线材整理
- 标记物品
- 包装加固

### 3. 螺丝类

建议准备：

- 十字螺丝刀
- 一字螺丝刀
- 精密螺丝刀套装
- 小扳手
- 六角扳手

用途：

- 设备外壳
- 支架
- 电池仓
- 家具小修

### 4. 测量类

建议准备：

- 卷尺
- 小直尺
- 记号笔
- 标签纸

用途：

- 标记位置
- 测量尺寸
- 记录编号
- 做库存标签

### 5. 防护类

建议准备：

- 工作手套
- 一次性手套
- 护目镜
- 口罩

用途：

- 防割伤
- 防灰尘
- 防飞溅
- 处理脏污物品

### 6. 清洁类

建议准备：

- 酒精棉片
- 干布
- 小刷子
- 棉签
- 密封袋

用途：

- 清洁接口
- 擦拭工具
- 收纳小零件
- 临时隔离污染物

## 操作步骤

1. 选择一个固定工具包。
2. 按类别分袋收纳。
3. 给工具包编号。
4. 写一张清单放在包内。
5. 每月按清单检查。
6. 使用后立即归位。
7. 缺少耗材及时补充。

## 检查点

每月检查：

- 刀片是否生锈
- 胶带是否失粘
- 扎带是否变脆
- 螺丝刀是否缺失
- 手套是否破损
- 记号笔是否能写
- 工具包是否在固定位置

## 常见错误

- 工具很多但找不到。
- 没有耗材。
- 用完不放回。
- 工具包太重，没人愿意拿。
- 小零件散落在包里。
- 没有手套和护目镜。

## 停止条件

出现以下情况时，应停止使用相关工具：

- 刀片松动
- 手柄破裂
- 电工工具绝缘损坏
- 工具严重生锈
- 手套破损
- 维修对象涉及高压、电池鼓包、燃气或结构承重
