# 充电线与接口检查

## 用途

介绍充电线、USB 接口、电源适配器和设备充电口的日常检查方法，减少断电或外出时因线材故障导致关键设备无法充电。

## 适用场景

适用于手机、移动电源、手电、学习终端、Field Terminal、Core 周边设备等需要 USB 或适配器充电的设备。

这篇内容用于避免关键时刻出现“电源还有，线坏了”的低级灾难。

## 准备材料

### 检查项目

### 1. 线材外观

检查：

- 外皮是否破损
- 是否露铜
- 接头是否弯折
- 是否有烧焦痕迹
- 是否异常发热
- 是否接触不良
- 是否需要特定方向才能充电

### 2. 接口状态

检查：

- USB-C 口是否松动
- Lightning 或其他接口是否变形
- 是否有灰尘或异物
- 是否有锈蚀
- 是否插入后晃动明显
- 是否充电断断续续

### 3. 电源适配器

检查：

- 是否发热异常
- 是否有异响
- 外壳是否裂开
- 插脚是否松动
- 输出功率是否适合设备
- 是否来自可靠来源

### 推荐备份

建议至少准备：

- USB-C 短线
- USB-C 长线
- USB-A 转 USB-C 线
- 设备专用线
- 多口充电器
- 车载或移动电源连接线
- 小型线材收纳包

## 操作步骤

### 核心原则

1. 充电系统包括设备、线、接口和电源头。
2. 线材是易耗品，需要备份。
3. 接口松动会导致充电失败或发热。
4. 破损线材可能带来短路和火灾风险。
5. 不同设备的接口规格要记录清楚。

1. 统计所有关键设备的接口类型。
2. 给常用充电线分类。
3. 每根线至少测试一台设备。
4. 把失效线材立即淘汰。
5. 给关键设备准备专用备用线。
6. 把线材和设备一起放入对应收纳袋。
7. 在库存系统记录接口类型和线材数量。

## 判断标准

### 检查点

每月检查：

- 关键设备是否能正常充电
- 备用线是否可用
- 接口是否松动
- 是否有破损线材
- 是否有重复但无用的旧线
- 是否缺少某类接口线

## 风险提示

### 常见错误

- 线材很多，但关键接口没有。
- 坏线不扔，和好线混在一起。
- 用劣质线给大功率设备充电。
- 线材放太散，停电时找不到。
- 只检查设备电量，不检查充电链路。
- 一根线承担全部关键设备。

### 停止条件

出现以下情况时，应停止使用线材或适配器：

- 露铜
- 烧焦味
- 接头融化
- 充电时明显发热
- 插头松动打火
- 设备反复断充
- 外壳破裂
- 接口变形
