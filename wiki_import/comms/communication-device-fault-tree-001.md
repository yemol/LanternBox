# 通信设备故障排查顺序

## 用途

用于通信设备无法连接、收不到、发不出或断续时的统一排查。顺序排查能减少误判，也能保护电量。

## 操作步骤

1. 记录设备编号、时间、地点、电量和故障现象。
2. 先查电源、电池、充电线和开关状态。
3. 再查接口、天线、耳机、音量、频道和设置。
4. 然后换安全位置或等待正确通信窗口。
5. 最后核对对端设备和备用路径。

## 判断标准

- 可继续：单一原因被确认并记录，恢复后能完成回执。
- 需降级：故障间歇出现，无法稳定复现。
- 停止条件：设备进水、发热、异味、电池异常或接口损坏时，不继续开机测试。

## 风险提示

- 不要同时换设备、频道、位置和天线。
- 不要把设备故障误判成人员失联。
- 不要消耗保底电量做反复测试。
