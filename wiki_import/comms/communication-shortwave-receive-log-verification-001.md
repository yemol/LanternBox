# 短波通信基础

## 用途

用于用短波接收外部信息时控制误听、谣言和耗电。这里以收听、记录、核验为主，不把短波当作稳定双向通信保证。

## 操作步骤

1. 设定固定收听时段和收听人。
2. 记录日期、时间、频率、位置、电量和内容摘要。
3. 把天气、交通、供水、医疗和安全信息分开记录。
4. 同一重要信息等待复听或另一来源印证。
5. 未核信息标为待核，不立即改变路线或配给。

## 判断标准

- 可继续：信息能被复述，频率和时间可查。
- 需复核：噪声大、只听到片段或来源不明。
- 停止条件：设备进水发热、内容无法辨认或消息要求高风险行动但未核实时，停止据此行动。

## 风险提示

- 不要凭噪声补全消息。
- 不要整夜开机耗尽电量。
- 不要把未核短波信息当作团队公告。
