# 无线电通信基本原理

## 用途

用于让小团队在使用对讲机、手持电台或简易接收设备前，先理解最小术语。这里不讨论复杂工程，只把频道、频率、监听、发射、接收、回执、噪声和窗口转成行动语言。

## 操作步骤

1. 把频道或频率写在纸上，不靠口头记忆。
2. 先听再发，确认当前信道没有被占用。
3. 消息只说对象、位置、状态、需求和下次窗口。
4. 收到消息后回读关键字段，避免只说“收到”。
5. 把失败、噪声、无人回应和电量变化写入通信记录。

## 判断标准

- 可继续：所有成员知道主频道、备用频道、监听窗口和回执方式。
- 需调整：有人把频道、频率、窗口或呼号混用。
- 停止条件：频道用途不明、身份不清或连续呼叫无人确认时，不继续占用信道。

## 风险提示

- 不要把标称距离当作现场可用距离。
- 不要在公开信道里说库存、住所、人员全名和路线细节。
- 无线电只是通信路径之一，失败时要转入纸条、集合点或人工传递。
