# 位置报告的导航字段

## 用途

用于外出者向留守人或队友报位置，避免只说模糊地点导致无法判断路线进度。

## 适用场景

- 外出队到达检查点或错过窗口。
- GNSS 坐标可用但需要配合地标。
- 留守方需要判断是否超时或偏离路线。

## 准备材料

- 通信日志
- 检查点表
- 坐标或地标
- 下一窗口

## 操作步骤

1. 先报人员和检查点编号。
2. 再报坐标或地标、方向和当前状态。
3. 说明是否按原路线、是否遇到危险点、是否需要改路。
4. 给出下一通信窗口和预期下一检查点。
5. 接收方回读关键字段并记录。

## 判断标准

- 合格报告包含人、位置、时间、状态、下一动作和回执。
- 需追问：只说“河边”“快到了”或没有方向。
- 停止条件：位置不明、身份不明或危险正在扩大时，停止普通任务推进。

## 风险提示

- 不要公开发送敏感住所和库存细节。
- 不要把未确认坐标当成已到检查点。

## 替代方案

- 无坐标时使用检查点编号、地标、方向和耗时。
- 通信失败时按错过检查点规则等待或回撤。

## 记录建议

记录发送人、接收人、时间、位置字段、回执和下一窗口。
