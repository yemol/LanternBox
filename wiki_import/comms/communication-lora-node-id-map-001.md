# LoRa 节点编号和位置图

## 用途

用于让团队知道每个 LoRa 节点在哪里、谁负责、用什么供电、覆盖哪个方向。位置图是故障排查和交接的基础。

## 操作步骤

1. 给每个节点写唯一编号。
2. 在纸图或简图上标出节点位置和备用位置。
3. 写下负责人、供电方式、上次测试时间和角色。
4. 标记固定节点、移动节点、中继节点和备用节点。
5. 每次移动节点后更新图，不只改口头说明。

## 判断标准

- 可继续：别人能按图找到节点并知道负责人。
- 需补记：图上只有位置，没有电量、角色和更新时间。
- 停止条件：节点身份不明、位置记录冲突或外人可能按图找到核心住所时，暂停使用该图并重画低暴露版本。

## 风险提示

- 不要用同一个编号标两个节点。
- 不要把敏感住所、库存和病人位置写在公开节点图上。
- 不要让移动节点随人外出后仍被当固定节点。
