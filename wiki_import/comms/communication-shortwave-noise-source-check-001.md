# 短波噪声来源初查

## 用途

用于短波收听时噪声太大、内容断续或似乎有多路信号重叠的初步判断。目标是区分可记录信息和待核信息。

## 操作步骤

1. 记录频率、时间、位置、电量和噪声类型。
2. 远离充电设备、逆变器、电机和金属遮挡。
3. 改变安全位置或天线方向，只改一个变量。
4. 等下一个收听时段复听同一频率。
5. 无法听清时标为待核，不补写缺失内容。

## 判断标准

- 可继续：同一内容能在较低噪声下复听。
- 需待核：只听到片段，或不同人复述不一致。
- 停止条件：噪声导致关键信息不清，且信息涉及撤离、配给或医疗时，不据此直接行动。

## 风险提示

- 不要把猜测写成事实。
- 不要在噪声很大时传播半截消息。
- 不要为换位置进入危险或潮湿区域。
