# 无线电短消息格式

## 用途

用于无线电、LoRa 或人工转述时统一消息结构。短消息不是少说关键内容，而是只保留行动所需字段。

## 操作步骤

1. 先写草稿：对象、时间、位置、状态、需求。
2. 删除闲聊、猜测和重复描述。
3. 加上下次联系窗口或返回时间。
4. 发送后要求对方回读位置、需求和时间。
5. 把是否确认写入通信日志。

## 判断标准

- 可继续：接收者能复述谁、在哪、需要什么、何时再联系。
- 需重发：对方只确认一部分字段，或把时间和地点听混。
- 停止条件：消息涉及敏感库存、住所核心位置或成员身份时，先改写为编号和约定地标。

## 风险提示

- 不要把长段解释塞进低速信道。
- 不要用“这里”“那边”“快点来”替代位置和任务。
- 不要把未经核实的消息标成确定信息。
