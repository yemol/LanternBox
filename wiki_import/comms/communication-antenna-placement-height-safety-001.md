# 天线高度影响

## 用途

用于选择天线或 LoRa 节点的临时位置。较高位置可能改善通信，但不能用攀爬、靠近电线或暴露人员来换取信号。

## 操作步骤

1. 先找稳定、干燥、可复查的位置。
2. 优先选择桌面、窗边内侧、屋檐下或安全高点。
3. 避开电线、金属大面、火源、睡眠区和通道。
4. 固定线缆，避免人员经过时拉倒设备。
5. 测试后记录位置、方向、成功率和异常。

## 判断标准

- 可继续：位置稳定，不绊人，不受雨水，能定期复查。
- 需调整：靠近金属遮挡、线缆横穿通道或儿童能拉到。
- 停止条件：需要攀爬不稳高处、靠近电线、雷雨接近或无法隔离人员时，停止架设。

## 风险提示

- 不要把天线架在门口、炉具旁或湿地面。
- 不要为提高信号离开团队约定安全范围。
- 不要让临时天线成为暴露住所的明显标记。
