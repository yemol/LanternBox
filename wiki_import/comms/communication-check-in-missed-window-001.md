# 错过通信窗口后的动作

## 用途

用于成员、外出队或节点错过固定通信窗口后的处理。错过窗口不等于立刻出事，必须按预案分级处理。

## 操作步骤

1. 记录错过对象、时间、频道和已尝试动作。
2. 检查本方设备、电量、天线和记录是否正确。
3. 等待下一个约定窗口或转备用频道。
4. 同时检查纸条点、集合点或人工传递记录。
5. 超过预设等待线后再启动本地核查。

## 判断标准

- 可继续：下个窗口恢复联系并完成回执。
- 需升级：连续错过、天气恶化或路线风险升高。
- 停止条件：没有返回路线、没有留守人员或需要进入高风险区域寻找时，不临时外出。

## 风险提示

- 不要因为一次错过窗口就多人外出。
- 不要把通信失败和人员遇险直接画等号。
- 不要忘记检查本方设备错误。
