# LoRa 节点部署位置

## 用途

用于选择 LoRa 节点的临时或半固定位置。好位置要同时考虑通信、供电、复查、安全和低暴露。

## 操作步骤

1. 先记录节点编号、负责人和用途。
2. 优先选择高处、干燥、少金属遮挡、可定期检查的位置。
3. 避开雨水、火源、通道、儿童可拿取点和明显外部可见点。
4. 做一次短消息测试，记录成功、延迟和电量。
5. 设定备用位置，避免一个点失效后临时乱移。

## 判断标准

- 可继续：节点稳定、能复查、能收发、位置不暴露核心住所。
- 需调整：电量难补、线缆容易受拉或人员会碰到。
- 停止条件：节点进水、发热、位置暴露、连续测试失败或无法安全取回时，停止作为关键节点。

## 风险提示

- 不要把节点放好后长期不查。
- 不要把节点放在金属箱内或湿地面。
- 不要让单一节点承担唯一关键联络。
