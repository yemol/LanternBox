# 天线连接检查

## 用途

用于无线电、LoRa 节点或短波接收设备使用前检查天线连接。天线异常不仅会造成失联，也可能让设备在发射时受损。

## 操作步骤

1. 先关闭发射，检查天线是否拧紧或插牢。
2. 查看接口是否潮湿、松动、裂开、变形或有腐蚀。
3. 检查线缆是否被夹、折、拉扯或外皮破损。
4. 轻轻固定线缆，避免重量直接拉在接口上。
5. 重新短测时记录位置、频道、电量和结果。

## 判断标准

- 可继续：天线牢固、接口干燥、线缆不受拉。
- 需调整：天线晃动、线缆绷紧或连接点容易被碰到。
- 停止条件：接口进水、金属变形、线缆破皮、天线断裂或设备发热时，不继续发射。

## 风险提示

- 不要无天线或天线松脱时发射。
- 不要用胶带掩盖接口进水问题。
- 不要让线缆穿过通道、门缝或火源附近。
