# 无线电和 LoRa 电量预算

## 用途

用于把通信设备纳入每日能源安排。通信电量要服务报平安、危险消息和外出联络，不应抢占照明、医疗或净水相关底线。

## 操作步骤

1. 列出手机、对讲机、LoRa 节点、收音机和备用电源。
2. 记录每台设备当前电量和负责人。
3. 把电量分成监听、短发射、收听、节点运行和备用。
4. 设定每日窗口，窗口外休眠或关机。
5. 每次窗口后记录电量变化和未确认消息。

## 判断标准

- 可继续：通信设备有保底电量和明确用途。
- 需降级：测试和闲聊消耗超过窗口预算。
- 停止条件：电量低于团队保底、设备快速掉电、发热或无法补电时，停止非紧急通信。

## 风险提示

- 不要长时间搜索信号。
- 不要让所有设备同时开机等待。
- 不要把通信测试排在照明和医疗电量之前。
