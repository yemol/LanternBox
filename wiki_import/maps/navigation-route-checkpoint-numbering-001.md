# 路线检查点编号

## 用途

用于外出路线、撤离路线和任务轨迹记录，帮助留守方知道外出者应该在哪里、何时报告。

## 适用场景

- 路线有多个岔路或危险点。
- 外出者需要按窗口报位置。
- FT path point 需要和纸图对应。

## 准备材料

- 路线卡
- 地图或草图
- 编号规则
- 通信日志

## 操作步骤

1. 从起点开始编号，不跳号。
2. 给转折点、危险点、补给点、等待点和返回点编号。
3. 每个检查点写预期到达时间或最晚报告时间。
4. 在位置报告中使用同一编号。
5. 路线变化时更新编号表，不口头改名。

## 判断标准

- 可用：编号能在纸图、FT 和通信日志中对应。
- 需修正：同一地点多个编号或同一编号多个地点。
- 停止条件：检查点编号混乱导致留守方无法判断超时时，暂停外出计划。

## 风险提示

- 不要只用地名，不用编号。
- 不要临时添加检查点却不通知留守人。

## 替代方案

- 没有地图时按地标顺序编号。
- 通信失败时用编号写纸条或返回后补记。

## 记录建议

记录编号、地标、预期时间、风险、报告状态和变更原因。
