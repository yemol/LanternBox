# 轨迹转返回摘要

## 用途

用于把 FT track 或手机轨迹变成纸质路线和口头交接，让没有设备的人也能理解。

## 适用场景

- 外出者返回后需要交接路线。
- 迷路时只剩轨迹片段和地标记忆。
- 留守人需要知道哪里危险、哪里可等。

## 准备材料

- 轨迹记录
- 纸笔
- 地图或草图
- 外出者复述

## 操作步骤

1. 先排除漂移段和断点不明段。
2. 选出起点、转折点、危险点、资源点、等待点和返回点。
3. 把每段写成地标、方向、预计耗时和风险。
4. 标明哪些点是设备记录，哪些是人工补充。
5. 让另一人按摘要复述返回路线。

## 判断标准

- 可用：摘要能在纸图上画出，并能被第二人复述。
- 需补充：只有坐标，没有地标和方向。
- 停止条件：关键转折点缺失或漂移未核时，不把摘要作为独立返回路线。

## 风险提示

- 不要把完整轨迹截图当作路线说明。
- 不要省略危险点和等待点。

## 替代方案

- 轨迹不足时使用检查点记录和外出者口述。
- 仍无法复述时进入迷路最近确定点规则。

## 记录建议

保存摘要版本、采用轨迹段、放弃段、复述人和复查时间。
