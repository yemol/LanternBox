# 离线地图版本管理

## 用途

用于出发前确认地图是否覆盖目标路线、返回路线和备用路线，不让地图文件只存在一个设备里。

## 适用场景

- 手机网络不可用，需要依赖离线地图。
- 多台设备地图版本不同。
- 路线跨出已下载区域。

## 准备材料

- 离线地图设备
- 纸质索引卡
- 地图覆盖范围记录
- 备用电源

## 操作步骤

1. 写下地图名称、区域、下载或更新时间。
2. 打开目标路线、返回路线和备用路线区域。
3. 确认地图能在无网络状态下显示。
4. 把地图版本编号写到外出路线卡。
5. 旧版本保留标记，不和当前版本混用。

## 判断标准

- 可用：目标、返回和备用范围都能离线打开。
- 需降级：地图缺一段路线、图层打不开或设备电量不足。
- 停止条件：地图覆盖范围不明且没有纸质草图时，不做远距离外出。

## 风险提示

- 不要假设在线地图缓存等于完整离线地图。
- 不要用不同版本地图口头争论路线。

## 替代方案

- 离线地图不可用时，用纸图、手绘草图和检查点缩短路线。
- 只有一台设备有地图时，出发前写纸质摘要。

## 记录建议

记录地图版本、覆盖范围、设备、检查人和失效日期。
