# GNSS 定位基础和限制

## 用途

用于使用 FT、手机或手持定位设备前，先判断当前位置是否有足够可信度。它不教复杂测量，只给现场记录和降级的边界。

## 适用场景

- 设备显示有坐标，但周围地标和地图不完全一致。
- 刚开机、刚出室内、在山谷或建筑旁，位置可能漂移。
- 需要把基地点、水源点、危险点或检查点交给他人。

## 准备材料

- 能显示 fix 状态的设备
- 纸笔
- 附近地标
- 时间记录方式

## 操作步骤

1. 先看设备是否显示 fix 或类似已定位状态，不把正在搜索卫星当作已定位。
2. 记录读数时间和所在环境，例如室内、林下、山谷、桥下或开阔地。
3. 用附近固定地标复核读数方向，不只看屏幕上的一个点。
4. 需要作为关键点时，隔一小段时间做第二次读数。
5. 把坐标和地标一起写下，避免只留下数字。

## 判断标准

- 可用：fix 稳定，位置和地标相符，两次读数没有明显跳动。
- 需复查：坐标跳动、方向不合地标、设备刚开机或环境遮挡明显。
- 停止条件：无 fix、位置连续跳变或无法说明地标时，不把该点作为集合点或返回依据。

## 风险提示

- 不要把一次读数当成绝对位置。
- 不要只发坐标不发地标。
- 不要在无 fix 时继续深入陌生路线。

## 替代方案

- 没有稳定坐标时，记录地标、方向、步数、时间和手绘草图。
- 设备电量不足时，优先保存基地点和最近确定点。

## 记录建议

记录坐标、时间、设备、fix 状态、地标、记录人和是否复查。
