# 返回路线规划

## 用途

用于任何离开驻点的行动。返回路线必须在出发前明确，而不是等迷路或天黑后才临时寻找。

## 适用场景

- 目标路线有岔路、河沟、坡地或夜间风险。
- 外出者可能需要提前折返。
- FT track 或纸图需要转成返回说明。

## 准备材料

- 路线卡
- 地图或草图
- 检查点编号
- 照明和时间余量

## 操作步骤

1. 先标原路返回线，再标备用返回线。
2. 写下最近安全点和不可继续的撤回线。
3. 为关键岔路设置返回方向标记。
4. 把返回所需时间按保守速度估算。
5. 目标没到但返回条件触发时，按返回线行动。

## 判断标准

- 可用：从目标、任一检查点和中途停止点都能说明怎么回。
- 需复查：只有去程路线，没有备用或最近安全点。
- 停止条件：返回线被切断、标记缺失或照明不足时，不继续向目标推进。

## 风险提示

- 不要把去程等同于可返回路线。
- 不要进入需要攀爬、涉水或穿越陌生地形的单向路线。

## 替代方案

- 备用路线不可用时退回最近确定点。
- 无法确认返回方向时停止移动并进入迷路处理。

## 记录建议

记录原路、备用路、最近安全点、撤回线和返回失败原因。
