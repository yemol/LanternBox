# 返回路线标记

## 用途

用于在外出取水、巡查、转移或搜索时建立可回撤路线，减少迷路、分队失联和夜间返回困难。

## 适用场景

适用于短距离、低风险、可留下标记的路线。不适用于暴雨、洪水、冲突、陌生夜路或需要穿越不稳定地形的行动。

## 准备材料

最低条件是路线草图、铅笔、统一方向标记和可识别地标。条件允许时准备布条、粉笔、反光片、编号卡、指南针和返回时间表。

## 操作步骤

1. 出发前在草图上画出起点、目标、转折点、危险点和最晚返回时间。
2. 选择不会移动的地标作为节点，例如大树、墙角、坡顶、桥头或明显石块。
3. 每到节点记录方向、距离或步数，并留下不易误读的返回标记。
4. 遇到岔路时标出已走方向和禁止方向，避免返回时误入新路。
5. 返回时逐项核对节点，发现标记缺失立即停止前进并回到上一个确定点。
6. 回来后修正草图，写明标记失效、路线难点和下次改进。

## 判断标准

- 正常情况：节点连续、方向清楚、返回时间可控，未同行的人能按草图理解路线。
- 异常情况：标记会被风雨移动、节点相似、岔路未说明、返回时间超出预期。
- 是否继续：下一节点可确认且有回撤点时继续；两个连续节点无法确认时返回最近确定点。

## 风险提示

- 风险来源：过度依赖记忆、临时改路和标记被移走。
- 停止条件：天气恶化、照明不足、队伍分散、标记消失或人员疲劳影响判断时停止前进。

## 替代方案

不能留下现场标记时，用步数、方位、地标顺序和草图记录。缺工具时用可复述的路线口诀，但回到驻点后立即写下。

## 记录建议

记录路线编号、节点、标记方式、耗时、失败点和最安全返回点。交接时说明哪些标记可能失效。
