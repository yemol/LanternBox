# 简易洗手点设置

## 用途

介绍停水、避难或户外场景下，如何设置一个低耗水、可持续、方便成员使用的简易洗手点。

## 适用场景

适用于停水、临时避难、户外工作、家庭隔离、成员腹泻或水资源有限但仍需保持手卫生的情况。

这篇内容用于保证关键场景下仍然能清洁手部。

## 准备材料

### 基础配置

需要：

- 清洁水容器
- 小出水口或可控倒水工具
- 肥皂
- 接水盆或排水容器
- 纸巾或专用干手布
- 垃圾袋
- 标签

可选：

- 脚踏或倾倒式水壶
- 免洗洗手液
- 一次性手套
- 小桌或架子

## 操作步骤

### 核心原则

1. 如厕后、处理食物前、处理伤口前后必须优先洗手。
2. 洗手点要固定，不能让成员到处找水和肥皂。
3. 少量流水比一盆多人共用水更安全。
4. 洗手点应远离饮用水和食物储存区。
5. 污水要有收集或排放方案。

### 1. 选择位置

洗手点应靠近：

- 厕所出口
- 食物处理区入口
- 医疗处理区入口
- 垃圾处理区出口

但不要直接放在食物和饮用水旁边。

### 2. 设置水流

优先使用小流量出水：

1. 水壶小口倒水。
2. 带水龙头的储水桶。
3. 打孔瓶临时出水。
4. 一人倒水，一人洗手。

避免多人反复把手伸进同一盆水里。

### 3. 放置肥皂

肥皂应固定在洗手点，避免丢失或沾满污物。

### 4. 处理污水

污水不能流到：

- 食物区
- 饮用水区
- 睡眠区
- 常走通道

污水容器满后应及时处理。

## 判断标准

### 检查点

每天检查：

- 是否还有清洁水
- 是否有肥皂
- 污水盆是否满
- 干手用品是否干净
- 洗手点是否被挪走
- 垃圾袋是否更换
- 成员是否知道位置

## 风险提示

### 常见错误

- 用一盆水让多人反复洗。
- 洗手点没有肥皂。
- 洗完手用脏毛巾擦。
- 污水流到饮水区。
- 洗手点设置太远，成员懒得用。
- 有腹泻成员后不加强手卫生。

### 停止条件

出现以下情况时，应立即调整洗手点：

- 洗手水被污染
- 污水溢出
- 肥皂缺失
- 干手布发臭
- 洗手点靠近食物
- 多人出现腹泻或呕吐
- 洗手区变成绊倒风险

## 记录建议

### 备注

壳中灯系统中，洗手点可记录为“避难空间配置”。

手卫生是最便宜的防线。资源不足时，越要把这条防线做得简单、固定、人人看得见。
