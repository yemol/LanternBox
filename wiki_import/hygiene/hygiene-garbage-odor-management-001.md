# 垃圾与异味管理

## 用途

介绍停水、停电、避难或垃圾清运不稳定时，如何分类、密封、隔离和记录垃圾，减少异味、虫害和疾病风险。

## 适用场景

适用于垃圾清运不稳定、停水、停电、临时避难、食物腐败、医疗废弃物或室内异味增加的情况。

这篇内容用于减少污染、虫害和感染风险。

## 准备材料

准备结实垃圾袋或带盖桶、扎带或绳子、标签、手套、吸水材料、清洁水、洗手用品、尖锐物临时容器和远离饮水食物的暂存位置。最低条件是能把湿垃圾、尖锐物、污染物和普通干垃圾分开。

## 操作步骤

### 核心原则

1. 垃圾要分类。
2. 潮湿垃圾和食物垃圾优先密封。
3. 医疗污染物要单独处理。
4. 垃圾区远离饮水、食物和睡眠区。
5. 异味是风险信号，需要记录和处理。

### 垃圾分类

建议分为：

- 干垃圾
- 湿垃圾
- 食物残渣
- 医疗污染物
- 尖锐物
- 电池和电子废弃物
- 可再利用材料
- 危险废弃物

### 1. 设置垃圾区

垃圾区应：

- 远离食物
- 远离饮用水
- 远离睡眠区
- 通风相对较好
- 不挡通道
- 便于清理

### 2. 密封湿垃圾

食物残渣、腐败物、呕吐物、污染纸巾等应：

1. 尽量沥干。
2. 放入小袋。
3. 扎紧。
4. 再放入外层垃圾袋。
5. 标记日期。

### 3. 处理医疗垃圾

如有血液、分泌物、污染纱布或手套：

1. 单独装袋。
2. 明显标记。
3. 不与普通垃圾混放。
4. 处理后洗手。
5. 避免儿童接触。

### 4. 控制异味

可以：

- 增加密封层
- 减少含水量
- 及时转移到安全外部位置
- 使用带盖桶
- 增加局部通风和异味复查
- 清洁垃圾桶表面

## 判断标准

### 检查点

每天检查：

- 是否有异味
- 是否有液体渗漏
- 是否有虫子
- 垃圾袋是否破损
- 医疗垃圾是否混入普通垃圾
- 食物垃圾是否及时密封
- 垃圾区是否靠近饮水区

## 风险提示

### 常见错误

- 所有垃圾混在一起。
- 湿垃圾不密封。
- 医疗污染物随手丢。
- 垃圾放在通道上。
- 腐败食物留在室内太久。
- 处理垃圾后不洗手。
- 电池和湿垃圾混放。

### 停止条件

出现以下情况时，应立即升级处理：

- 垃圾液体渗漏
- 明显腐败恶臭
- 出现大量虫害
- 医疗污染物暴露
- 尖锐物刺破垃圾袋
- 垃圾区靠近饮水或食物
- 成员出现腹泻、呕吐等卫生相关症状

## 替代方案

没有垃圾袋时，用带盖桶、旧容器或多层纸布临时收纳，并把湿垃圾加吸水材料后封口；没有手套时用塑料袋隔手并立即清洁双手。垃圾已渗漏或靠近饮水食物时，先隔离和清洁地面，再重新分区。

## 记录建议

### 备注

壳中灯系统中，垃圾异常可记录为“卫生风险”。

建议记录：

- 日期
- 垃圾类型
- 异味等级
- 是否渗漏
- 是否虫害
- 处理方式

垃圾管理不是琐事。它是避难空间能否长期住下去的底层卫生防线。
