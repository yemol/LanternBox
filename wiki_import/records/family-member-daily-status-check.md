# 家庭成员每日状态检查

## 适用场景

适用于避难、停电、疾病观察、长期居家、外部压力较高、成员分工协作或资源不足的情况。

这篇内容用于每天快速确认每个人是否还处于可行动、可沟通、可照护状态。

## 核心原则

1. 每天至少检查一次成员状态。
2. 重点看饮水、进食、睡眠、精神、体温和行动能力。
3. 异常要记录，不要只靠记忆。
4. 老人、儿童、慢性病患者和情绪低落成员要重点关注。
5. 状态检查不是审问，而是照护。

## 检查项目

### 1. 基础身体状态

记录：

- 是否发热
- 是否头晕
- 是否腹泻
- 是否呕吐
- 是否疼痛
- 是否受伤
- 是否能正常行动

### 2. 饮水和进食

记录：

- 今天喝了多少水
- 是否有明显口渴
- 是否有尿
- 是否吃过饭
- 是否食欲明显下降
- 是否有特殊饮食需求

### 3. 睡眠和疲劳

记录：

- 昨晚睡了多久
- 是否频繁醒来
- 是否极度疲劳
- 是否白天嗜睡
- 是否需要轮班调整

### 4. 情绪和沟通

观察：

- 是否沉默异常
- 是否烦躁
- 是否恐慌
- 是否反应迟钝
- 是否能正常交流
- 是否需要陪伴或休息

## 操作步骤

1. 固定每天一个时间检查。
2. 用同一套问题快速询问。
3. 对高风险成员单独记录。
4. 发现异常立即写入日记。
5. 安排补水、休息、饮食或照护。
6. 第二天复查是否改善。

## 推荐记录模板

```text
日期：
成员：
体温：
饮水：
进食：
睡眠：
精神状态：
行动能力：
异常：
处理：
复查时间：
```

## 检查点

每天检查：

- 是否有人明显少喝水
- 是否有人没吃饭
- 是否有人持续疲劳
- 是否有人情绪失控
- 是否有人发热
- 是否有人受伤
- 是否有人需要减少任务

## 常见错误

- 只关注物资，不关注人。
- 认为成年人会自己说。
- 忽视沉默和退缩。
- 不记录轻微异常。
- 让身体不适成员继续承担重任务。
- 每天检查问题不一致，无法比较趋势。

## 停止条件

出现以下情况时，应立即升级照护：

- 意识混乱
- 持续高热
- 呼吸困难
- 严重脱水
- 反复呕吐
- 持续腹泻
- 情绪崩溃或无法沟通
- 无法行走
- 伤口恶化
- 老人、儿童或慢性病患者状态明显变差

## 备注

壳中灯后续成员系统应支持每日状态卡片。

人是系统的核心资源。Core 保存知识，但每天状态检查是在保存使用知识的人。
