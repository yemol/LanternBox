# 物资分配冲突的安全风险

## 用途

帮助团队识别物资分配争执何时会从意见不合变成安全风险，并把冲突从库存区、火源区和门口转移开。

## 适用场景

适用于水、食物、药品、电池、保暖物资或工具分配时出现争吵、抢拿、指责或不信任。

不适用于已经有人动手、抢夺、堵门或威胁伤害的情况；此时先分离人员和保护弱势成员。

## 准备材料

- 分配记录
- 暂停口令
- 库存负责人
- 冷静区
- 复盘表

## 操作步骤

1. 先暂停发放，不在争吵中继续打开库存。
2. 把人员从库存区、火源、门口和工具区分开。
3. 由记录人读出现有数量和既定规则，避免凭情绪争辩。
4. 先保障儿童、老人、病人和执行关键任务者的最低需求。
5. 冲突结束后修改分配规则，而不是只要求大家忍耐。

## 判断标准

- 争执降到可听清事实，可恢复讨论。
- 有人靠近库存抢拿、拍打物品或堵住出口，应停止分配。
- 规则不透明导致反复争吵，应改为公开清点。

## 风险提示

- 不要在库存箱打开时争论。
- 不要让最激动的人负责发放。
- 不要用羞辱语言压制需求表达。

## 替代方案

无法达成一致时，先按最低维持量临时发放，并设下一次固定复核时间。

## 记录建议

记录争执原因、涉及物资、暂停时间、最终规则和是否需要分开保管。
