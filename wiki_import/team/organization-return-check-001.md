# 归队检查为什么要固定流程

## 用途

本条用于处理归队检查的固定流程，让团队在长期断供、小团队自持和离线环境中能做出一致判断。它的边界是把“归队检查为什么要固定流程”拆成可执行的检查、停止线和记录项，避免和同组其它条目混成一个笼统提醒。

## 适用场景

- 适用于外出取水、探路、搬运、接触陌生环境后返回居住点。
- 重点服务于组织相关的现场判断、交接和复查。
- 不适用于本条不允许外出者直接进入睡眠、厨房或物资核心区。

## 准备材料

- 归队点
- 检查表
- 洗手/清洁物
- 物资袋
- 异常记录

## 操作步骤

1. 在固定归队点先点名和确认伤病。
2. 检查鞋底、手部、衣物、容器和工具是否污染或破损。
3. 交接路线、遇到的人、危险点和物资变化。
4. 需要清洁或隔离的物品先留在边界区。
5. 归队后再更新任务板和库存表。

## 判断标准

- 人员完整、状态清楚、污染没有带入厨房和睡眠区。
- 路线信息已给到下一次外出者。
- 物资数量与领取/归还记录一致。
- 停止条件：本条不允许外出者直接进入睡眠、厨房或物资核心区。

## 风险提示

- 不要让疲劳外出者一回来就做饭或抱儿童。
- 不要把陌生水源容器直接放进干净区。
- 这是中风险条目；出现异常、疲劳或信息不明时，先暂停并让第二人复核。

## 替代方案

- 没有独立归队点时用门口一块垫子和固定桌面做边界。
- 时间紧时先做人员、伤口、污染三项。

## 记录建议

记录归队时间、人员状态、携回物、污染处理、路线风险和复查。
