# 灰烬进入土壤前转交判断

## 用途

用于 Waste 和 Agriculture 的接口。Waste 负责把冷灰按来源分好；Agriculture 决定能不能进入土壤、进入哪里、用多少和何时复查。

## 操作步骤

1. 只处理已冷却确认的灰烬。
2. 分开干净木灰、混合燃料灰和疑似污染燃料灰。
3. 来源不明、塑料/涂料/垃圾燃烧灰不进入食用地块。
4. 想入土前贴标签并转交 Agriculture Guide。
5. 记录来源、火源日期和转交人。

## 判断标准

- 来源清楚。
- 没有混入垃圾燃烧灰。
- 进入土壤前有农业二次判断。

## 风险提示

停止条件：灰烬来源不明、气味异常、混入塑料或化学燃料、扬尘接近食物饮水时，不进入农业循环。

## 替代方案

无法确认来源时封存或丢弃，不作为土壤材料。

## 记录建议

记录灰烬来源、冷却确认、农业交接和是否禁用。
