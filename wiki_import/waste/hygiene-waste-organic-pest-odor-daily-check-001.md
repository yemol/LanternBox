# 有机废弃物虫害异味日查

## 用途

用于长期断供下对厨余、湿垃圾和堆肥等待桶做每日巡查。问题早发现，可以减少苍蝇、鼠害、异味和厨房污染。

## 操作步骤

1. 每天固定时间检查封盖是否完整。
2. 看桶底、袋口和地面是否渗漏。
3. 看是否有苍蝇、虫卵、鼠迹和咬痕。
4. 闻到明显臭味时先加密封和移远，不继续堆积。
5. 记录异常和处理人。

## 判断标准

- 封盖完整。
- 无渗漏扩散。
- 虫鼠和臭味没有进入厨房、饮水和睡眠区。

## 风险提示

停止条件：渗液、虫鼠聚集、臭味突然加重、袋体破裂或病人剩饭混入时，停止继续加入新厨余。

## 替代方案

容器不足时减少每日厨余堆积量，优先密封高异味和高渗漏批次。

## 记录建议

记录检查时间、异常类型、处理动作和下一次复查。
