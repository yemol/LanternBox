# FT-02 外壳原型模块尺寸记录 v1.0

项目：LanternBox / 壳中灯\
设备：FT-02 Field Terminal Prototype

## 1. 设计原则

-   第一版目标：外场测试原型
-   优先可靠性、可维护性、模块替换能力
-   不追求极限厚度
-   模块尽可能独立安装
-   一个螺丝刀即可完成主要维护

# 3. 墨水屏模块

## Waveshare 4.26 inch E-Paper HAT

参数：

-   分辨率：800 × 480
-   显示区域：92.8 × 55.8 mm
-   裸屏厚度：0.926 mm

连接：

    VCC  -> 3V3
    GND  -> GND
    DIN  -> GPIO11
    CLK  -> GPIO12
    CS   -> GPIO10
    DC   -> GPIO9
    RST  -> GPIO3
    BUSY -> GPIO14
    PWR  -> GPIO18

实测结构高度：

    裸屏 + HAT板 + 排针 + 杜邦线

    总高度：25mm

# 5. LoRa通信模块

型号：

Heltec WiFi LoRa 32 V3

参数：

  项目       数据
  ---------- ---------------------------
  主控       ESP32-S3FN8
  LoRa芯片   SX1262
  USB芯片    CP2102
  Flash      8MB
  频率       470\~510MHz / 863\~928MHz
  输出功率   21±1dBm
  灵敏度     -136dBm

尺寸：

    50.2 × 25.5 × 10.2 mm

接口：

-   USB Type-C
-   3.7V电池接口
-   IPEX 1.0天线接口
-   2.54mm Header

结构要求：

-   保留外接天线能力
-   天线区域避免金属遮挡
-   避免紧贴ESP32-S3天线

------------------------------------------------------------------------

# 6. GNSS定位模块

型号：

u-blox MAX-M10S-00B-01

支持：

-   GPS
-   北斗 BDS
-   GLONASS
-   Galileo

尺寸：

    13.1 × 15.7 mm

接口：

  引脚   功能
  ------ ----------
  VCC    电源
  GND    地
  TX     UART发送
  RX     UART接收
  PPS    秒脉冲

结构要求：

-   靠近外壳顶部
-   保证天空方向
-   避免金属遮挡
-   保留独立模块仓
