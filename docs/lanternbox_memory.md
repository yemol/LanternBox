# LanternBox / 壳中灯 / 火种计划 项目记忆

## 1. 项目定位

LanternBox 是完全离线、封闭、自有网络、自主供电的个人/团队生存与知识系统。

核心愿景：

> 为人类的存续留下一个小小的火种。

设计原则：

- 无外部支援假设
- 能源优先
- 最小可用能力优先
- 通讯端到端加密
- 默认离线
- 本地知识库与本地 AI 结合

## 2. 系统结构

### Core

Core 负责：

- 离线知识库
- Wiki
- RAG / 本地模型
- 任务调度
- 数据归档
- 权限与安全
- 能源监控
- 本地 AI 助手
- 应急指南库
- 终端同步

### Field Terminal

现场终端，负责：

- 现场记录
- 拍照 / 录音
- GPS / 北斗
- 环境传感器采集
- LoRa / Wi-Fi / 蓝牙 / 有线同步
- 低功耗应急能力

### Study Terminal

学习终端，负责：

- 离线阅读
- 手写笔记
- 语音笔记
- 教学模式
- 与 Core 同步学习资料

### Sensor Node

传感器节点，负责：

- 气温
- 湿度
- 气压
- 电量
- 时间
- 坐标
- LoRa / 低功耗通信状态

## 3. 当前软件栈

当前阶段软件栈：

- FastAPI 后端
- 静态前端页面
- PocketBase Wiki
- Ollama 本地模型
- 本地 RAG / Wiki 检索
- 应急指南库
- 日常记录
