# Guide Schema V1

Guide 是 LanternBox 的“行动卡”。它回答的是：现在该怎么做、先做什么、做到什么程度停止或升级。Guide 应短、清晰、可执行，适合在压力、停电、断网和多人协作场景中直接使用。

## 字段定义

- `id`：稳定编号，例如 `DG-0547`。编号一旦发布不复用。
- `title`：行动卡标题，使用“对象 + 场景 + 行动方向”的短标题。
- `category`：知识域或业务分类，例如 `水`、`维修`、`通讯`。
- `domains`：检索使用的标准领域数组，例如 `water`、`medical`、`power`。
- `priority`：优先级，建议使用 `P0`、`P1`、`P2`。
- `risk_level`：风险等级，必须使用 `normal`、`caution`、`high`、`critical` 之一。
- `scenario`：适用场景，说明什么时候使用这张卡。
- `goal`：目标，说明完成后应该降低什么风险或达成什么状态。
- `keywords`：检索关键词数组，包含用户口语、对象词、风险词、动作词和常见别名。
- `tools`：建议工具或材料。没有工具时写可替代物。
- `steps`：可执行步骤，按顺序写，避免长篇解释。
- `check`：完成检查项，帮助用户确认是否做到位。
- `common_mistakes`：常见错误，特别是会扩大风险的动作。
- `fallback`：条件不足时的替代方案数组；即使只有一项也必须使用数组。
- `stop_or_escalate`：停止尝试、放弃物品、撤离、求援或升级处理的信号。
- `related_wiki`：关联的真实 Wiki slug 数组，用于解释原因和判断边界；必须与 Wiki `guide_links` 双向对称。
- `notes`：补充说明、适用限制、维护者提醒。

## 编写原则

- 每张 Guide 只解决一个明确场景，不做百科解释。
- 优先写“先做什么”和“不要做什么”。
- 不把攻击性武器制作或针对人的伤害性装置作为主线。
- 维修类 Guide 必须包含断电、停用、承重、火源、割伤、污染等停止线。
- 野外食物获取类 Guide 必须强调法律、人道、安全、误伤、卫生和替代方案。
- `keywords` 要能覆盖用户口语，例如“还能用吗”“泡过水”“没信号”“发臭”。

## Risk Level 定义

- `normal`：常规记录、检查、整理和低风险维护。
- `caution`：错误操作可能造成资源损失、轻伤、污染扩散或任务失败。
- `high`：错误操作可能造成严重伤害、中毒、火灾、电击、感染、饮用水污染或团队安全风险。
- `critical`：正在发生或高度可能发生生命危险、火灾爆燃、严重中毒、失温/中暑、严重出血、不可控暴力，或必须立即停止行动的场景。

`caution`、`high`、`critical` 必须具备非空 `stop_or_escalate`。`high`和 `critical` 还必须明确写出停止、禁用、隔离、撤离、停用、不可入口、不可通电或不可继续等边界。

## 完整示例

```json
{
  "id": "DG-0547",
  "title": "燃气味或疑似泄漏：禁止明火与先撤离",
  "category": "火源 / 保温 / 通风 / 一氧化碳风险",
  "domains": ["security"],
  "priority": "P0",
  "risk_level": "critical",
  "scenario": "厨房、室内、燃气罐附近或管线周边闻到燃气味、煤气味、液化气味，或怀疑存在泄漏。",
  "goal": "避免点火、开关电器和近距离通信引发爆炸风险，先通风、撤离和隔离火源。",
  "keywords": [
    "燃气味",
    "煤气",
    "液化气",
    "泄漏",
    "点火",
    "明火",
    "蜡烛",
    "酒精炉",
    "厨房",
    "爆炸风险",
    "不要点火",
    "禁止明火"
  ],
  "tools": [
    "干净布条",
    "门窗",
    "纸质联络卡",
    "手电"
  ],
  "steps": [
    "闻到燃气味或疑似泄漏时，不要点火查看，不点蜡烛、火柴、打火机、酒精炉或燃气灶。",
    "不要开关灯、不要插拔电器、不要启动排风扇，不在泄漏点附近打电话。",
    "如果安全可行，轻动作关闭燃气阀或液化气罐阀；如果阀门附近味道很重，先撤离。",
    "打开门窗自然通风，人员离开厨房和低洼积气位置。",
    "到室外、安全、上风或开阔位置后，再联系家人、物业或专业人员。"
  ],
  "check": [
    "所有明火已熄灭。",
    "没有人在泄漏点附近操作电器。",
    "人员已离开燃气味最重的位置。",
    "已在安全位置沟通下一步。"
  ],
  "common_mistakes": [
    "点火确认是否漏气。",
    "打开排风扇快速排气。",
    "在厨房内拨打电话。",
    "继续使用蜡烛或酒精炉照明。"
  ],
  "fallback": [
    "无法关闭阀门时，优先撤离人员。",
    "没有手电时，使用远离泄漏点的低风险照明，不进入气味浓重区域。",
    "无法联系专业人员时，保持人员远离并持续观察气味变化。"
  ],
  "stop_or_escalate": [
    "气味迅速变重。",
    "有人头晕、恶心或呼吸不适。",
    "听到持续漏气声。",
    "已出现火焰、爆燃、浓烟或人员受伤。"
  ],
  "related_wiki": [
    "safety-gas-leak-001",
    "safety-gas-leak-002"
  ],
  "notes": "燃气味场景下，最危险的是为了确认而点火或操作电器。"
}
```
