# LanternBox Knowledge Coverage V1

LanternBox 知识库定位为“长期生存能力库”，不是单纯的应急指南库。它要支持离线环境下的个人、家庭或小团队，在资源紧张、外部支持不稳定、信息受限的情况下，完成持续判断、行动、复盘和知识传承。

知识覆盖按优先级分为：

- P0：直接影响存活、安全和基本秩序的核心能力，优先补 Guide 行动卡。
- P1：支撑数周到数月的持续生活、协作和低强度生产能力。
- P2：支撑长期知识保存、环境理解、生产恢复和重建的广域能力。

## P0 核心生存能力

### 水

- 典型场景：饮用水不足、水质浑浊或有异味、停水、储水容器不足、高温缺水、雨水或可疑水源是否能用。
- Guide 应覆盖的行动卡：饮用水优先级、缺水时用途排序、储水容器临时替代、煮沸和过滤流程、污染水源停止使用条件。
- Wiki 应覆盖的精选知识：脱水信号、水质风险、煮沸边界、过滤和消毒差异、每日最低饮水需求。
- Kiwix/ZIM 应补充的广域知识：水文基础、常见水源污染、简易过滤材料、公共卫生供水知识。
- 第一批建议条目：缺水 24 小时行动卡、家庭储水纪律、水质异常判断、雨水收集边界、高温下补水和盐分观察。
