# Wiki / 手册内容补强优先级计划

## 目标

在当前阶段，优先补强能直接支撑应急问答的本地 Wiki / 手册内容，减少对 ZIM 背景资料的依赖。

## 优先级 P0：必须先补

### 1. 医疗与脱水
- 老人发热脱水判断
- 儿童发热与脱水观察
- 小伤口清洁与感染升级信号
- 烫伤、擦伤、鼻出血、扭伤等基础处理边界

### 2. 水与饮用水
- 水源污染判断
- 煮沸与过滤的边界
- 容器降级与非饮用标记
- 水短缺时的优先级排序

### 3. 电与火风险
- 充电宝鼓包与发热边界
- 插线板进水与漏电风险
- 火灾初期撤离边界
- 燃气味与点火禁忌

### 4. 食品安全
- 变质食品判断
- 储存与分配优先级
- 交叉污染与污染区隔离

## 优先级 P1：后续补强

- 设备维修与临时替代
- 通信与安全协同
- 疫情/卫生管理
- 灾害转移与避难

## 交付原则

- 每个条目都必须说明：适用场景、禁止事项、升级信号、记录建议。
- 文章应优先服务“判断 + 降级 + 记录”，而不是单纯提供背景知识。
- 需要持续让检索系统能命中这些内容，而不是只是存档在文件夹里。
